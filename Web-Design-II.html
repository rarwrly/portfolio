Web Design II

The main project for Web Design II was to create a news style website for a subject of our choice. For my subject I chose the Yuru-Chara of Japan. In Japan most prefectures, sports teams and major businesses have a mascot. The requriements of this website were to create a multi-page website with a minium of three sections and with a minium of fifteen articles spread scross the website. This web site was also made to be responsive for a phone screen and to implement some javascript for the menu drop downs.
